Página de Áudio

A tag <audio> em HTML permite incorporar arquivos de áudio em páginas web. Ela oferece controles para reproduzir, pausar e ajustar o volume do áudio diretamente no navegador.

Dentro da tag <audio>, utilizamos múltiplas tags <source> para fornecer diferentes formatos de áudio, garantindo maior compatibilidade entre navegadores.

Se o navegador não suportar nenhum dos formatos fornecidos, é possível exibir uma mensagem alternativa entre as tags <audio> e </audio>.
